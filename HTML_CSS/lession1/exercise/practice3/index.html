Bắt đầu một session trong PHP

Trước tiên, bạn cần khởi tạo session trong PHP bằng cách sử dụng session_start(). Điều này thường được thực hiện ở đầu trang PHP trước khi gửi bất kỳ nội dung nào đến trình duyệt.

Khi sử dụng $_SESSION, bạn cần khởi tạo session bằng cách gọi hàm session_start(). Điều này thường được thực hiện ở đầu file PHP.

    <?php

        session_start();

          if(isset[$_SESSION[counter]]){

            $_SESSION[counter] += 1;

          } else {

                $_SESSION[counter] = 1;
                
          }
              
   ?>
